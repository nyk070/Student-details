# Student Management System

## Project Overview

This is a beginner-friendly Python project that helps practice
fundamental programming concepts. Users can enter student information,
calculate marks-related statistics, and view results in a structured
format.

## Objective

Practice core programming concepts: - Variables and data types - User
input and type conversion - Arithmetic, comparison, and logical
operators - Built-in functions (`max()`, `min()`)

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

## Features

-   Enter student name and marks for 3 subjects
-   Calculate total, average, highest, and lowest marks
-   Compare marks using comparison operators
-   Check if student passed all subjects

## How to Run

1.  Open the notebook in Jupyter or Colab
2.  Run all cells
3.  Enter inputs when prompted

## Concepts Covered

-   Variables & Data Types
-   User Input & Type Conversion
-   Arithmetic, Comparison & Logical Operators
-   Built-in Functions
-   Formatted Output
